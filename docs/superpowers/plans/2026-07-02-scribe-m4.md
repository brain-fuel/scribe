# scribe M4 (v0.4) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** `icon` preset toolkit (Apple grid, size ladder, one-call icon sets) and `gui` surface layer (Blit interface for any windowing toolkit, plus a zero-dep animated-GIF writer as the testable motion surface).

**Architecture:** `icon` composes existing pieces: Spec -> RoundRect plate -> Canvas, with the Apple radius ratio and size ladder as data. `gui` stays a surface, not a framework: a one-method `Surface` interface any toolkit satisfies, a doc-comment example for ebiten integration (real windowing needs cgo/display, untestable here), and `WriteGIF` driving `draw func(frame int) dl.Program` through the dl interpreter into stdlib GIF encoding (Plan9 palette + Floyd-Steinberg, both deterministic).

**Tech Stack:** Go 1.26, stdlib only. Same global constraints as M1-M3 (zero deps, determinism, no trig, no em-dashes or en-dashes in prose, the usual commit trailer).

## Environment honesty

A real window cannot be opened or tested in this environment (headless WSL, no display, and any toolkit is a cgo dependency). M4 therefore ships the integration point (`gui.Surface`) and the deterministic motion target (GIF), and documents toolkit integration in the package comment. A live example belongs to a later milestone run on a machine with a display.

### Task 1: icon package

**Files:** `icon/icon.go`, `icon/icon_test.go`

**Interfaces:**
- `icon.AppleRadiusRatio = 0.2237`, `icon.AppleSizes = []int{16, 32, 64, 128, 256, 512, 1024}`
- `icon.Spec{Size int; Radius float64; Style path.CornerStyle; Fill color.RGBA}` (Radius <= 0 means AppleRadiusRatio * Size)
- `icon.Plate(s Spec) *scribe.Canvas`
- `icon.WriteSet(dir, name string, base Spec, sizes []int) ([]string, error)` writing `<name>_<size>.png` per size (radius rescaled per size when base.Radius <= 0; when explicit, scaled proportionally from base.Size), returning paths in size order.

Tests: plate center opaque and corner transparent at 64; default radius = ratio*size (compare against a direct RoundRect render byte-for-byte); WriteSet writes all ladder files decodable at right sizes; explicit radius scales proportionally (radius 10 at size 100 renders identically to radius 20 at size 200 downchecked structurally: corner pixel classification matches).

Implementation sketch:
```go
func Plate(s Spec) *scribe.Canvas {
	r := s.Radius
	if r <= 0 {
		r = AppleRadiusRatio * float64(s.Size)
	}
	c := scribe.NewCanvas(s.Size, s.Size)
	sz := float64(s.Size)
	c.Fill(path.RoundRect(geom.RectXYWH(0, 0, sz, sz), r, s.Style), s.Fill)
	return c
}

func WriteSet(dir, name string, base Spec, sizes []int) ([]string, error) {
	var out []string
	for _, sz := range sizes {
		sp := base
		if sp.Radius > 0 && base.Size > 0 {
			sp.Radius = base.Radius * float64(sz) / float64(base.Size)
		}
		sp.Size = sz
		p := filepath.Join(dir, fmt.Sprintf("%s_%d.png", name, sz))
		if err := Plate(sp).SavePNG(p); err != nil {
			return nil, err
		}
		out = append(out, p)
	}
	return out, nil
}
```

### Task 3: gui package (Surface + WriteGIF)

**Files:** `gui/gui.go`, `gui/gui_test.go`

**Interfaces:**
```go
// Surface is anything that can display a frame.
type Surface interface {
	Blit(img *image.RGBA)
}
type GIFOptions struct {
	W, H    int
	Frames  int
	DelayCS int // per-frame delay, centiseconds
}
func WriteGIF(w io.Writer, o GIFOptions, draw func(frame int) dl.Program) error
```
Per frame: `scribe.NewCanvas` -> `dl.Render(draw(i), c)` -> quantize to `image.Paletted` with `palette.Plan9` via `draw.FloydSteinberg.Draw` -> append to `gif.GIF` with the delay -> `gif.EncodeAll`. Deterministic end to end.

Package comment documents toolkit integration:
```go
// Integrating with a windowing toolkit is one Blit call per frame.
// With ebiten (github.com/hajimehoshi/ebiten), for example:
//
//	func (g *game) Draw(screen *ebiten.Image) {
//	    c := scribe.NewCanvas(w, h)
//	    dl.Render(g.frame(), c)
//	    screen.WritePixels(c.Image().Pix)
//	}
//
// scribe stays zero-dependency; the toolkit lives in your module.
```

Tests: WriteGIF of a 3-frame moving dot decodes (`gif.DecodeAll`) to 3 frames of right size with right delays; two renders byte-identical; a type assertion that a test double satisfies Surface.

Commit: `feat: gui surface interface and deterministic GIF animation`
